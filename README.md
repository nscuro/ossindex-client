*Unofficial Go client for Sonatype OSS Index*

## Installation

```
GO111MODULE=on go get github.com/nscuro/ossindex-client
```

## Compatibility

|  Go   | OSS Index REST API |
| :---: | :----------------: |
| 1.15+ |         v3         |
